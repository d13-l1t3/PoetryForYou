\subsection*{Retrospective}

\textbf{What went well:}
\begin{enumerate}
    \item Voice search works end-to-end after fixing the ``searching'' stage.
    \item All deployment issues resolved (DB healthcheck, Whisper cache, openai version).
    \item Architecture well-documented with PlantUML diagrams.
\end{enumerate}

\textbf{What didn't go well:}
\begin{enumerate}
    \item Whisper transcription is slow on CPU (~3-5s).
    \item Google Custom Search API returns errors (invalid argument).
    \item Solo development means no real code review.
\end{enumerate}

\textbf{Action points:}
\begin{enumerate}
    \item Try Whisper ``tiny'' model for faster transcription.
    \item Add CD (continuous deployment) to the pipeline.
\end{enumerate}

% ==================== QUALITY ATTRIBUTES ====================
\section{Quality Attribute Scenarios}

Full document: \href{https://github.com/d13-l1t3/PoetryForYou/blob/main/docs/quality-assurance/quality-attribute-scenarios.md}{docs/quality-assurance/quality-attribute-scenarios.md}

Three ISO 25010 sub-characteristics chosen and confirmed with the customer:

\subsection*{1. Performance Efficiency --- Time Behavior}

\begin{tabularx}{\textwidth}{lX}
\toprule
\textbf{Stimulus} & User sends a text command \\
\textbf{Measure} & Response time $\leq$ 500ms \\
\textbf{How achieved} & FastAPI async, in-memory poem search, indexed DB queries \\
\bottomrule
\end{tabularx}

\subsection*{2. Security --- Confidentiality}

\begin{tabularx}{\textwidth}{lX}
\toprule
\textbf{Stimulus} & Attacker inspects repo or compromises container \\
\textbf{Measure} & Zero secrets in code; container read-only, non-root \\
\textbf{How achieved} & \texttt{.env} in \texttt{.gitignore}, \texttt{bandit} in CI, \texttt{read\_only: true} \\
\bottomrule
\end{tabularx}

\subsection*{3. Usability --- Accessibility}

\begin{tabularx}{\textwidth}{lX}
\toprule
\textbf{Stimulus} & User sends voice message in \texttt{/search} mode \\
\textbf{Measure} & Correct poem in top 3 for $\geq$85\% of clear queries \\
\textbf{How achieved} & Whisper STT + LLM keyword extraction + scored search ranking \\
\bottomrule
\end{tabularx}
